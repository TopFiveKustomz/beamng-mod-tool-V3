# BeamNG Mod Health Tool V3

A browser-based mod analyzer and auto-fix tool for **0** mods.

It detects JBeam issues, validates mod structure, repairs common errors, and exports clean ZIP files directly in your browser. No installation required.

---

## 🚀 Features

- Multi-vehicle detection in ZIP files
- JBeam structure validation (nodes, slots, files)
- Door / hood / trunk hinge warnings
- Wheel alignment sanity checks (useful for custom rims like 26s builds)
- Strict Mode (BeamNG repository-style validation)
- Auto-fix system (safe naming + slot repair)
- Clean ZIP export with correct vehicle naming

---

## 📦 How to Use

1. Open the website (GitHub Pages link)
2. Upload a BeamNG mod ZIP
3. Click **Analyze**
4. Review issues + score
5. Click **Fix & Export**
6. Download cleaned mod ZIP

---

## ⚙️ Optional Settings

- **Strict Mode**  
  Enforces BeamNG folder structure (`vehicles/` validation)

- **Vehicle Override Name**  
  Manually set or correct detected vehicle name

---

## 🧠 What It Fixes

- Donor references (bruckell / mohawk)
- Broken slot types
- Basic missing node detection
- Folder structure issues
- Incorrect naming exports

---

## ⚠️ What It Does NOT Fix

- Broken suspension geometry
- Incorrect wheel physics setup
- Advanced JBeam node errors
- Blender-level pivot issues

---

## 🛠️ Ideal Use Cases

- Mesh swap workflows (Malibu, GTO, etc.)
- Custom wheel builds (22s–26s setups)
- Mod QA before in-game testing
- Cleaning third-party mods
